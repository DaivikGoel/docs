## Dashboard Layout

The Shor merchant dashboard is your central hub for managing payment links, viewing transactions, and monitoring your business metrics.

## Main Sections

### 1. Overview Page

The overview page displays:
- **Recent Transactions**: Your latest payments with key details
- **Saved Payment Links**: Quick access to your active payment links
- **Quick Stats**: Total volume, number of transactions, etc.

### 2. Navigation Menu

Located on the left sidebar:

<Tabs>
  <Tab title="Overview">
    Dashboard home with summary metrics and recent activity
  </Tab>
  <Tab title="Payment Links">
    - View all payment links

  <Tab title="Transactions">

  <Tab title="Create Link">

  </Tab>
  <Tab title="Settings">
    - Profile configuration

  </Tab>
</Tabs>

## Key Features

### Payment Link Cards

Each payment link displays:
- **Title & Description**: What you're selling
- **Amount**: Fixed or variable pricing
- **Status**: Active/Inactive indicator
- **Created Date**: When the link was created
- **Quick Actions**:
  - Copy link
  - View QR code
  - Edit details
  - Toggle active status

### Transaction Table

The transaction view includes:

| Column | Description |
|--------|-------------|
| Transaction ID | Unique identifier for each payment |
| Customer | Wallet address or identifier |
| Amount | Payment amount in USD |
| USDC Received | Actual USDC amount received |
| Date & Time | When the transaction occurred |
| Status | Pending, Completed, or Failed |
| Payment Link | Which link was used |
| Transaction Hash | Blockchain verification link |

### Real-time Updates

<Note>
The dashboard updates in real-time as transactions occur. No need to refresh!
</Note>

## Dashboard Actions

### Creating Payment Links
Click the "Create Payment Link" button to:
1. Set payment details
2. Configure fee handling
3. Generate shareable link

### Managing Links
For each payment link, you can:
- **Copy**: Get the link URL instantly
- **Share**: Generate QR codes or share directly
- **Edit**: Update amounts, descriptions, or images
- **Disable**: Temporarily stop accepting payments

### Viewing Transactions
- **Filter**: By date range, status, or payment link
- **Search**: Find specific transactions by ID
- **Export**: Download transaction data as CSV
- **Details**: Click any transaction for full details

## Keyboard Shortcuts

Speed up your workflow with these shortcuts:

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + K` | Quick search |
| `Cmd/Ctrl + N` | Create new payment link |
| `Cmd/Ctrl + C` | Copy selected link |
| `Cmd/Ctrl + /` | Show all shortcuts |
