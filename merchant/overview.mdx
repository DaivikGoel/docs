## Key Features

### Payment Link Cards

Each payment link displays:
- **Title & Description**: What you're selling
- **Amount**: Fixed or variable pricing
- **Status**: Active/Inactive indicator
- **Created Date**: When the link was created
- **Quick Actions**:
  - Copy link
  - View QR code
  - Edit details
  - Toggle active status

### Transaction Table

The transaction view includes:

| Column | Description |
|--------|-------------|
| Transaction ID | Unique identifier for each payment |
| Customer | Wallet address or identifier |
| Amount | Payment amount in USD |
| USDC Received | Actual USDC amount received |
| Date & Time | When the transaction occurred |
| Status | Pending, Completed, or Failed |
| Payment Link | Which link was used |
| Transaction Hash | Blockchain verification link |

## Dashboard Actions

### Creating Payment Links
Click the "Create Payment Link" button to:
1. Set payment details
2. Configure fee handling
3. Generate shareable link

### Managing Links
For each payment link, you can:
- **Copy**: Get the link URL instantly
- **Share**: Generate QR codes or share directly
- **Edit**: Update amounts, descriptions, or images
- **Disable**: Temporarily stop accepting payments

### Viewing Transactions
- **Filter**: By date range, status, or payment link
- **Search**: Find specific transactions by ID
- **Export**: Download transaction data as CSV
- **Details**: Click any transaction for full details

## Settings

Current settings options:

1. **Webhooks**: Configure URL for transaction notifications
2. **Test Webhooks**: Send test payloads to verify integration
